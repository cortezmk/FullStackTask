# Instructions

After project is cloned please use attached PS file.

to setup database:

```
./volue setup
```
to build images:
```
./volue build
```
and to run project:
```
./volue run
```
Once project is running service can be found here:
```
http://0.0.0.0:5000/apidocs/
```
to stop running services:
```
./volue stop
```
